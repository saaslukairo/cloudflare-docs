_Cloudflare Pipelines scalar function implementations are based on
[Apache DataFusion](https://arrow.apache.org/datafusion/) (via [Arroyo](https://www.arroyo.dev/)) and these docs are derived from
the DataFusion function reference._

## `encode`

Encode binary data into a textual representation.

```
encode(expression, format)
```

**Arguments**

- **expression**: Expression containing string or binary data

- **format**: Supported formats are: `base64`, `hex`

**Related functions**:
[decode](#decode)

## `decode`

Decode binary data from textual representation in string.

```
decode(expression, format)
```

**Arguments**

- **expression**: Expression containing encoded string data

- **format**: Same arguments as [encode](#encode)

**Related functions**:
[encode](#encode)
