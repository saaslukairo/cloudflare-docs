The following literals are supported:

| Type          | Syntax                                                                                                   |
| ------------- | -------------------------------------------------------------------------------------------------------- |
| integer       | `42`, `-42`                                                                                              |
| double        | `4.2`, `-4.2`                                                                                            |
| string        | `'so long and thanks for all the fish'`                                                                  |
| boolean       | `true` or `false`                                                                                        |
| time interval | `INTERVAL '42' DAY`<br/>Intervals of `YEAR`, `MONTH`, `DAY`, `HOUR`, `MINUTE` and `SECOND` are supported |
