## Component

:::caution
This component is automatically included on pages that have a `reviewed` frontmatter entry, under the page title.
:::

```mdx live
import { LastReviewed } from "~/components"

<LastReviewed
    date="2024-07-01"
/>
```

## Frontmatter

```mdx
---
title: My Tutorial
reviewed: 2024-07-01
---
```
