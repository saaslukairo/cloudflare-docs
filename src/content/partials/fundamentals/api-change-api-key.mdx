If your API key might be compromised, change your API key:

1. Log in to the Cloudflare dashboard.

    <DashButton url="/?to=/:account/home" />

2. Go to **My Profile** > **API Tokens**.
3. In the **API Keys** section, find your key.
4. Select **Change**.
